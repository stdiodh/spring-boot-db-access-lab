# 영속성 저장과 계층 분리 체크리스트

## 학생 체크리스트

- [ ] 메모리 저장과 DB 저장 차이를 설명할 수 있습니다.
- [ ] `PostEntity`가 테이블과 연결된다는 점을 설명할 수 있습니다.
- [ ] `PostRepository`가 어떤 역할을 하는지 말할 수 있습니다.
- [ ] `Controller -> Service -> Repository -> DB` 흐름을 설명할 수 있습니다.
- [ ] 생성 / 조회 / 수정 / 삭제가 DB에 반영되는 것을 확인할 수 있습니다.
- [ ] Swagger와 MySQL 조회 도구를 이용해 저장 결과를 확인할 수 있습니다.
- [ ] 관계 매핑이 왜 필요한지 입문 수준에서 설명할 수 있습니다.
- [ ] N+1이 어떤 상황에서 생기는지 말할 수 있습니다.

## 강사 / 발표 / PPT 체크리스트

- [ ] 메모리 저장과 DB 저장 차이를 한 장면으로 비교할 수 있는가
- [ ] Entity, Repository, Service, Controller 역할 그림이 있는가
- [ ] CRUD 흐름이 어디서 DB 저장으로 바뀌는지 코드로 보여줄 수 있는가
- [ ] MySQL 조회 화면 또는 CLI 결과를 시연할 수 있는가
- [ ] 관계 매핑과 N+1을 구현 범위와 구분해서 설명할 수 있는가
- [ ] 다음 시퀀스의 Validation/Exception 필요성을 연결할 수 있는가
