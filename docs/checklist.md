# 인증과 JWT 체크리스트

## 학생 체크리스트

- [ ] 인증과 인가의 차이를 설명할 수 있습니다.
- [ ] 회원가입 시 비밀번호를 왜 인코딩해야 하는지 설명할 수 있습니다.
- [ ] 로그인 성공 후 access token이 응답으로 내려오는 것을 직접 확인했습니다.
- [ ] 토큰 없이 `/auth/me`를 호출해 401을 직접 확인했습니다.
- [ ] 토큰을 넣고 `/auth/me`를 호출해 현재 사용자 email을 직접 확인했습니다.

## 강사 / PPT 체크리스트

- [ ] 회원가입 -> 로그인 -> JWT -> `/auth/me` 흐름 그림이 있는가
- [ ] `encode(...)`와 `matches(...)` 차이를 짧게 설명할 자료가 있는가
- [ ] 공개 API와 보호된 API를 같은 화면에서 비교 시연할 수 있는가
- [ ] 토큰 없음 401과 토큰 있음 200을 바로 비교할 수 있는가
- [ ] 다음 시퀀스의 Google OAuth 확장으로 자연스럽게 연결할 수 있는가
