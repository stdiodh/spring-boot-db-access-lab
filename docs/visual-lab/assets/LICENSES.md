# Visual Lab asset licenses

- `diagrams/*.svg`: A&I 프로젝트 자체 제작 자산입니다. 재사용 조건은 이 저장소의 라이선스 정책을 따릅니다. 저장소에 별도 라이선스가 없다면 이 문서가 독립적인 재사용 권한을 부여하지 않습니다.
- `icons/*.svg`: 기존 저장소 자산 `system-icons.svg`에서 분리한 자산이며 원본과 같은 저장소 라이선스 정책을 따릅니다.
- `system-icons.svg`: 기존 자산으로서 원래 저장소 정책과 출처 상태를 유지합니다.
- `visual-lab-mark.svg`: A&I 프로젝트 자체 제작 자산이며 이 저장소의 라이선스 정책을 따릅니다.
