One-time credential

메일 링크의 token으로 새 비밀번호를 설정합니다.

204 No Content

token 원문은 화면이나 저장소에 표시하지 않습니다. 서버는 hash·만료·사용 여부를 한 transaction에서 확인합니다.

앞뒤 공백도 비밀번호 원문의 일부이며 8~64자를 사용합니다.

주소에서 reset token을 회수했습니다. 비밀번호를 변경하면 같은 token은 다시 사용할 수 없습니다.
